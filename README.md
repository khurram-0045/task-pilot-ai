# TaskPilot AI 🚀

## a. App Name, Description & Problem Statement
* **App Name:** TaskPilot AI
* **What it Does:** A comprehensive, intelligent productivity and task management application designed to help users streamline their daily workflows, organize projects, track habits, and leverage smart AI guidance.
* **The Real Problem it Solves:** Modern students and professionals struggle with task overload, scattered notes, and lack of prioritization. TaskPilot AI solves this by centralizing tasks, project vaults, and habit tracking into a single, cohesive, distraction-free interface with built-in intelligent assistance.
* **Target Audience:** Students, developers, and remote workers looking for an efficient, lightweight personal productivity manager.

## b. Live Deployed URL
* [View TaskPilot AI Live on Vercel](https://task-pilot-ai-drab.vercel.app)

## c. Features List
* **Smart Dashboard:** Real-time overview of active tasks, habit streaks, and daily productivity metrics.
* **Task & Project Management:** Create, categorize, prioritize, and track tasks and multi-step projects effortlessly.
* **Habit Tracker:** Build consistent positive routines with interactive streak counters.
* **Secure Vault:** Store important snippets, notes, and documents locally and securely.
* **AI Assistance:** Built-in intelligent assistant screen providing contextual task breakdowns and productivity guidance.

## d. The AI Feature & System Instructions
* **What it Does:** The AI assistant analyzes user productivity challenges and provides structured, step-by-step breakdowns or prioritization strategies.
* **System Prompt / Instructions Used:** 
  > "You are TaskPilot AI, an expert productivity assistant. Your goal is to help the user break down complex tasks, prioritize daily goals efficiently, and maintain high productivity without feeling overwhelmed. Keep responses concise, clear, and actionable."

## e. Tools, Services & Technologies Used
* **Framework:** Next.js (React)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **State Management:** Zustand / Local State Storage
* **Deployment:** Vercel
* **Version Control:** Git & GitHub
